## Como executar o Projeto :

Primeiramente Clone o repositório

```bash
git clone https://github.com/zinoLath/ProjetoFinalByron.git
```

Segundamente instale as depêndencias do projeto:

```bash
npm install
```

E então executar com :

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
